# Web Helper 多端应用用户手册

## 概述

Web Helper是一个多端统一架构的应用程序，支持在Web、Electron和浏览器扩展三种环境中运行，提供一致的功能体验。

## 功能特性

### 核心功能
- 平台信息获取：识别当前运行环境（Web/Electron/浏览器扩展）
- 计数器功能：在不同平台间保持一致的计数体验
- 消息传递：跨平台的消息通信机制

## 各平台使用说明

### Web版本
1. 构建Web应用：`npm run build:all`
2. 访问Web应用：构建完成后访问dist目录下的index.html
3. 主要功能：
   - 查看平台信息
   - 使用计数器功能
   - 测试消息传递

### Electron版本
1. 启动Electron应用：`npm run dev:electron`
2. 主要功能与Web版本一致，但具有桌面应用特性

### 浏览器扩展版本
1. 构建扩展：`npm run dev:extension`
2. 安装扩展：
   - 打开Chrome浏览器
   - 访问chrome://extensions/
   - 开启开发者模式
   - 点击"加载已解压的扩展程序"
   - 选择`apps/extension/dist`目录
3. 使用扩展：
   - 点击浏览器工具栏中的扩展图标
   - 在弹出窗口中使用功能

## 常见问题

### 扩展安装问题
Q: 安装扩展时出现错误怎么办？
A: 请确保您已开启开发者模式，并选择正确的dist目录。

### 功能不一致问题
Q: 不同平台上功能体验不一致怎么办？
A: 请检查各平台的构建是否最新，并确保没有平台特定的错误。
